In order to get this game to open in eclipse:
1) Open Eclipse
2) File -> New Java Project
3) Put in a name for the Project
4) Click "Finish"
5) A popup asking if you want to create a module or not, say "No"
6) Copy all the files in this src/ folder and paste them into the src folder in the new java project you just made
7) Go back to eclipse and refresh the project

In order to run the program -> Run Game.java
